# AI-ML-Internship-Phase-2-Task-2
# Customer Churn Prediction Using Machine Learning

## Overview
This project predicts customer churn using machine learning techniques. Multiple classification algorithms are trained and compared to identify customers likely to leave a service.

## Problem Statement
Customer churn significantly impacts business revenue. Organizations need predictive models to identify at-risk customers and implement retention strategies before customers leave.

## Objectives
- Preprocess customer data.
- Handle numerical and categorical features.
- Train multiple classification models.
- Compare model performance.
- Identify the most effective model for churn prediction.

## Algorithms Used
- Logistic Regression
- Random Forest Classifier

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Workflow
1. Data Loading
2. Data Cleaning
3. Feature Engineering
4. Data Preprocessing
5. Train-Test Split
6. Model Training
7. Hyperparameter Tuning using GridSearchCV
8. Model Evaluation
9. Performance Comparison
10. Model Saving

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score
- Confusion Matrix

## Results
Both Logistic Regression and Random Forest models were evaluated. Random Forest generally demonstrated stronger predictive performance due to its ability to capture complex relationships within the data.

## Key Insights
- Customer behavior patterns strongly influence churn.
- Proper feature preprocessing improves performance.
- Ensemble methods can outperform linear models.
